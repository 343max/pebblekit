Logging
Foundation

Detailed Description

Functions related to logging from apps.

This module contains the functions necessary to log messages through Bluetooth.

Note
It is necessary to enable app logging output from the "settings->about" menu on the Pebble for them to be transmitted. Logging over Bluetooth is a fairly power hungry operation that non-developers will not need when your apps are distributed.
